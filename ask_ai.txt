# 라즈베리파이 5 CSI 카메라 애플리케이션 개발 요청 내역

1. 초기 요청 (2024-05-07)
- raspberrypi5에 원격접속해서 module3 csi 카메라를 이용해서 실시간 카메라 디스플레이와 파일캡쳐 파일저장을 할 수 있는 파이썬코드 gui 앱을 작성
- README.md에 개발과정을 자세히 기록
- requirements.txt 작성
- git으로 개발 단계마다 커밋과정 자동 기록

2. 한글화 요청 (2024-05-07)
- 모든 내용을 한글로 변경
- README.md와 camera_app.py의 모든 텍스트를 한글로 변경

3. 기능 개선 요청 (2024-05-07)
- 이미지 캡처 시 파일 저장 다이얼로그박스가 나타나도록 변경
- 라즈베리파이 csi 카메라 모듈3의 촛점 조절 기능 추가

4. 촛점 조절 방식 변경 요청 (2024-05-07)
- 슬라이더 조절 방식에서 자동 촛점 조절 방식으로 변경

# 구현된 주요 기능
1. 실시간 카메라 미리보기
2. 자동 촛점 조절
3. 이미지 캡처 및 저장 (파일 저장 다이얼로그 포함)
4. 한글 인터페이스
5. 원격 접속 지원
